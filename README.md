
## Requisitos
De preferencia, se necesita tener instalado **Visual Studio Code (VS Code)** para poder editar y ejecutar este proyecto de la forma más rápida y sin errores.

## Instalación y Ejecución

1. Descargar los archivos del proyecto y guardarlos en una carpeta en su computadora, ya sea como archivo .zip o clonando el repositorio en una carpeta de su prefrencia con git Bash,
    copiando y pegando el siguiente comando: git clone https://github.com/Davis-st/Proyecto-de-c-tedra.git
2. Abrir esa carpeta usando **Visual Studio Code**.
3. En VS Code, en apartado de Extensiones  buscar e instala una llamada **Live Server**.
4. Una vez instalada, abrir el archivo `index.html` en el editor.
5. Haz clic derecho en cualquier parte del código y selecciona la opción **"Open with Live Server"**.
6. El proyecto se abrirá automáticamente en su navegador web
7. Otra forma mas sencilla es visitando el siguiente enlace: http://tinyurl.com/proyectoped
## Tecnologías Utilizadas

* **HTML5:** Estructura de la interfaz.
* **CSS3:** efectos y animaciones. Todo en código puro, sin frameworks externos.
* **JavaScript:** Lógica básica de interacción.
